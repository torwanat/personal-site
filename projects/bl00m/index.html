bl00m - an artificial flower that opens when it detects light.

Powered by Raspberry Pi Pico and built with 3D-printed components, it offers a clean and complete design with a natural look. The assembly is easily accessible and highly modular.

Back to projects overview

Overview

bl00m is a result of group work involving mechanical design, electronics and programming. The flower opens and closes based on light intensity, which it detects with a sensor. The Raspberry Pi Pico microcontroller reads the sensor values and moves the servo motor accordingly. The design of the flower is modular, allowing for simple assembly and maintenance. It also allows easy access to the internal components, including batteries and a hidden on/off switch. The petals can be easily unscrewed and replaced or adjusted. From outside, the flower looks natural and elegant, with a smooth opening and closing motion.


  • Areas: 3D design, Electronics, Embedded Programming
  • Tech: Raspberry Pi Pico, C, 3D Printing, Laser Cutting
  • Status: On display at Fab Lab Oulu

Tech stack and architecture

The project runs on a Raspberry Pi Pico that reads values from a photoresistor and drives a servo that controls the petals. Since the sensor readings are analog, they are converted to digital values using the Pico's ADC. The resulting voltage level is then checked against set thresholds, and if a change is detected, the servo is moved to the corresponding angle. The servo operates a pulley system, which connects it to the bud using nylon strings running through the stem. A simple spring mechanism located in the bud ensures that the petals return to the correct position when tension is released. The whole system is powered by a small batter pack, which is hidden inside the pot. Thanks to the design, it is easy to take out the whole flower assembly and acess it and the internal electronics. The exterior is hand painted to look natural, with petals being made from farbed textile. This combined with the use of a real flower pot gives the whole assembly a natural and elegant look.

Implementation details

The implementation of the project involved overcoming several challenges, both mechanical and electronic.


Bud mechanism

To ensure the petals return to their original position when the tension is released, a tension mechanism was put into the bud. Making it small enough to fit and not be visible proved to be a challenge, and several prototypes were tested before settling on the final design. To make the movement even smoother, moving parts were lubricated with a small amount of sewing machine oil. Final design can be found on github.


Storing state data

To ensure that the flower maintains its state even after a power loss, the current state is stored in memory. This allows the flower to resume its previous state when power is restored. The implementation uses the Raspberry Pi Pico's built-in flash memory to store the state data, picking a chunk of memory that is not used by the program itself. The state data is written to flash memory whenever a change in state is detected, and read from flash memory on startup to restore the previous state:

Sensor calibration

Since the photoresistor is an analog sensor, a simple circuit had to be created to convert the analog signal to a digital one and make sure the voltage levels are within the expected range. This was achieved using a simple resistor and Pico's built-in ADC. The sensor was calibrated to ensure that the flower opens and closes at the desired light intensity levels.
